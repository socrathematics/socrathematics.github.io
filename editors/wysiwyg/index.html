    • How to use the WYSIWYG Editor

    This editor is located at https://cms.ymath.io/. You can use it without an account but you will not be able to save your work to the course cloud.

    Enabling Github

    You must have write access to socrathematics/socrathematics.github.io to use this feature. If you do not, you must save the html, fork the repository, upload your file, and submit a pull.

    1.  Go to Github, and go to Settings > Developer Settings > Personal Access Token
    2. Generate an access token with repository access. This means checking the first box, which will automatically check other boxes.
    3. Follow the instructions given to you to attach this token to your Course Maker account. (Yes, you need a separate account.)
    Note: Github can be enabled at any time, but you must have a Course Maker account to save your work.

    Writing

    • What can this editor do? This very page was written and published entirely using the editor. I never wrote a single line of code or opened Github.
    • How do I preview the course? If this isn't enough for you, hit the eye at the top.
    • How do I add navigation links? Highlight the navigation text, and hit the link button.
    • I did something weird. How do I edit the code? Hit the "<>" button at the top, and edit away!
    • How do I type math? Type it as you would normally in a LaTeX editor, and then hit the $x^2$ at the top.

    Publishing

    1. Write your course, just like this.
    2. Save your work using the cloud icon. (Must be logged in. Accounts are free.)
    3. Go to My Files > My Courses > > Submit.
    4. Wait for admin approval.
    5. If your fine was "returned", fix anything highlighted and resubmit.
    6. If your file was "approved", publish it! (Contact me for help.)
